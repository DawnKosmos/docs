Transfers funds between the different sections of an individual's account (not between subaccounts). For example, between the spot and derivatives accounts.
To send a transfer which is account type & parent/sub agnostic, use the Create Universal Transfer endpoint.

:::info
Please note that this interface supports Contract Account, Spot Account, Investment Account. The transferIn and transferOut account types cannot be the same.
You may transfer USDT from **SPOT** ONLY after upgraded to Unified Margin Account.
:::

#### HTTP Request
POST <code>/asset/v3/private/transfer/inter-transfer</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|transferId |<b>true</b> |string |UUID, which is unique across the platform |
|coin |<b>true</b> |string |Currency type |
|amount |<b>true</b> |string |Exchange to amount |
|fromAccountType |<b>true</b> |string |Account type |
|toAccountType |<b>true</b> |string |Account type |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|transferId |string |UUID, which is unique across the platform |
