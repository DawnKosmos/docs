Subscribe position stream to get position data change in real-time.

**Topic:** `position`

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|id |string |Message id|
|topic |string |Topic name |
|creationTime |number |Data created timestamp (ms) |
|data |array |Object |
|> category |string |Product type. `linear`,`option` |
|> symbol |string |Symbol name |
|> side |string |Position side. `Buy`,`Sell` |
|> size |string |Position size |
|> [positionIdx](../../enum#positionidx) |integer |Used to identify positions in different position modes|
|> tradeMode |integer |Trade mode. `0`: cross margin, `1`: isolated margin. Always `0` under unified margin account |
|> positionValue |string |Entry position value |
|> riskId |integer |Risk limit id |
|> riskLimitValue |string |Risk limit value corresponding to riskId|
|> entryPrice |string |Entry price |
|> markPrice |string |Mark price |
|> leverage |string |Leverage |
|> positionBalance |string |Please ignore |
|> positionMM |string |Position maintenance margin |
|> positionIM |string |Position initial margin |
|> liqPrice |string |Est.liquidation price. `""` for unified margin account |
|> bustPrice |string |Est.bankruptcy price. `""` for unified margin account |
|> tpSlMode |string |Tp/Sl mode. `Full`,`Partial` |
|> takeProfit |string |Take profit price |
|> stopLoss |string |Stop loss price |
|> trailingStop |string |Trailing stop |
|> unrealisedPnl |string |Unrealised profit and loss |
|> cumRealisedPnl |string |Cumulative realised PnL |
|> positionStatus |string |Position status |
|> createdTime |string |Position created timestamp (ms) |
|> updatedTime |string |Position data updated timestamp (ms) |
