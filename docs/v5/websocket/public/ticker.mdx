Subscribe the ticker data.<br/>
:::note
* This topic utilises the snapshot field and delta field. If response param is not found in the message, then its value has no change.
* Spot tickers message has **snapshot** ONLY
:::
Push frequency: **100ms**

**Topic:**<br/>
`tickers.{symbol}`
