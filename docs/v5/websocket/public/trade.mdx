Subscribe recent trades data in Bybit.<br/>
After subscription, you will be pushed trade message in real-time once there is an order filled.

Push frequency: **real-time**

**Topic:**<br/>
`publicTrade.{symbol}` **note**: option uses baseCoin, e.g., publicTrade.BTC

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|topic |string |Topic name |
|type |string |Data type. `snapshot` |
|ts |number |The timestamp (ms) that system generates the data |
|data |array |Object |
|> T |number |The timestamp (ms) that the order is filled |
|> s |array |Symbol name  |
|> S |string |Side. `Buy`,`Sell` |
|> v |string |Trade size |
|> p |string |Trade price |
|> [L](../../enum#tickDirection) |string |Direction of price change  |
|> i |string |Trade Id |
|> BT |boolean |Whether it is a block trade order or not|

### Response Example

```json
{
    "topic": "publicTrade.BTCUSDT",
    "type": "snapshot",
    "ts": 1672304486868,
    "data": [
        {
            "T": 1672304486865,
            "s": "BTCUSDT",
            "S": "Buy",
            "v": "0.001",
            "p": "16578.50",
            "L": "PlusTick",
            "i": "20f43950-d8dd-5b31-9112-a178eb6023af",
            "BT": false
        }
    ]
}
```
