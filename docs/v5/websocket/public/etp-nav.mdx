Subscribe leverage token nav stream

**Push frequency:** 300ms

**Topic:**<br/>
`lt.{symbol}` e.g.,lt.BTC3SUSDT [确认symbol的写法，是BTC3S还是BTC3SUSDT]

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|topic |string |Topic name |
|type |string |Data type. `snapshot` |
|ts |integer |The timestamp (ms) that system generates the data |
|data |array |Object |
|> time |integer |The generated timestamp of nav |
|> symbol |string |Symbol name |
|> nav |integer |Net asset value |
|> basketPosition |string |Total position value = basket value * total circulation |
|> leverage |string |Leverage |
|> basketLoan |string |Basket loan  |
|> circulation |string |Circulation |
|> basket |string |Basket |
