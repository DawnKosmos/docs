Get the trading fee rate

:::caution
Frequency limit, only 5 queries per minute
:::


#### HTTP Request
GET `/v5/account/fee-rate`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |category. spot, linear, inverse, option|
|symbol |false |string |symbol|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |category. spot, linear, inverse, option|
|list > symbol |string |symbol|
|list > takerFeeRate |string |taker fee rate|
|list > makerFeeRate |string |maker fee rate|
