Turn on/off auto-add-margin for **isolated** position

### HTTP Request
POST `/v5/position/set-auto-add-margin`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Product type. `linear`. **note**: USDC contract is not applicable |
|symbol |false |true |Symbol name |
|autoAddMargin |true |string |Turn on/off. `0`: off. `1`: on |
|positionIdx |false |string |Used to identify positions in different position modes. For one-way mode position, this param is **required**<li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li>|
