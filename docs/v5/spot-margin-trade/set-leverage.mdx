GET `/v5/spot-margin-trade/set-leverage`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|leverage |**true**| int |leverage|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status |string |1:success |
