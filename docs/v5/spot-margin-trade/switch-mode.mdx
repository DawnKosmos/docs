Switch Mode in cross margin

#### HTTP Request
GET `/v5/spot-margin-trade/switch-mode`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|switch |**true** |int |1:on，0:off |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|switchStatus |string |switch status（1:on，0:off）|
