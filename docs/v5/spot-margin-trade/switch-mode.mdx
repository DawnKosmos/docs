全仓杠杆开关

#### HTTP Request
GET `/v5/spot-margin-trade/switch-mode`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|switch |**true** |int |1:开启，0:关闭 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|switchStatus |string |全仓杠杆状态（1:开启，0:关闭）|
