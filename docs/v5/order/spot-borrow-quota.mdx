Query the qty and amount of borrowable coin in spot account.
> **Cover: Spot (Unified Account)**

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP Request
GET `/v5/order/spot-borrow-check`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `spot`|
|symbol |**true** |string |Symbol name|
|side |**true** |string |Transaction side. `Buy`,`Sell` |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|------ |
|symbol |string |Symbol name |
|side |string |Side |
|maxBorrowableQty |string |The maximum coin qty of borrowable<li>If spot margin trade on and symbol is margin trading pair, it returns available balance + max.borrowable amount</li><li>Otherwise, it returns actual balance</li>  |
|maxBorrowableAmount |string |The maximum coin amount of borrowable<li>If spot margin trade on and symbol is margin trading pair, it returns available balance + max.borrowable amount</li><li>Otherwise, it returns actual balance</li> |
|borrowCoin |string |Borrow coin |

### Request Example

```http
GET /v5/order/spot-borrow-check?category=spot&symbol=BTCUSDT&side=Buy HTTP/1.1
Host: api-testnet.bybit.com

```
