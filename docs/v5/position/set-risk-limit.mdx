Set risk limit. Click <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001164">here</a> to learn more about risk limit.
> **Cover: Linear Perpetual / Inverse Perpetual / Inverse Futures**

:::tip
Set the risk limit of the position. You could get risk limit information of each symbol [here](../market/risk-limit#).
:::

### HTTP Request
POST `v5/position/set-risk-limit`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `linear`,`inverse` |
|symbol |**true** |string |Symbol name|
|riskId |**true** |integer |Risk limit id|
|[positionIdx](../enum#positionidx) |**true** |integer |Used to identify positions in different position modes<li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li>|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|riskId |integer |Risk limit id |
|riskLimitValue |string |The position limit value corresponding to this risk id |
