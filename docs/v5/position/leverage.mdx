Set the leverage.

#### HTTP Request
POST `/v5/position/set-leverage`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Products category. `linear`,`inverse` |
|symbol |**true** |string |Symbol name|
|buyLeverage |**true** |string |(0, max leverage of corresponding [risk limit](../market/risk-limit#) |
|sellLeverage |**true** |string |(0, max leverage of corresponding risk limit]|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|retCode |integer |Success/Error code |
|retMsg |string |Success/Error message |
|result |Object | |
|time |integer |Timestamp |
