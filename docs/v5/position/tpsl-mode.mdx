Set TP/SL mode to Full or Partial
> **Covers: Linear contract (Unified Account)**

:::info
For partial TP/SL mode, you can set the TP/SL size smaller than position size.
:::

### HTTP Request
POST `/v5/position/set-tpsl-mode`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `linear`, USDT perpetual ONLY|
|symbol |**true** |string |Symbol name|
|tpSlMode |**true** |string |TP/SL mode. `Full`,`Partial` |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|tpSlMode |string  |`Full`,`Partial` |

### Request Example

```http
POST /v5/position/set-tpsl-mode HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672279325035
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json

{
    "symbol": "XRPUSDT",
    "category": "linear",
    "tpSlMode": "Full"
}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "tpSlMode": "Full"
    },
    "retExtInfo": {},
    "time": 1672279322666
}
```
