申赎记录查询

:::caution
API权限:读写权限
:::


#### HTTP Request
POST `/v5/spot-lever-token/order-record`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |**true** |string |LT资产名称，例如BTC3L |
|orderId |false	|string	|订单ID |
|startTime |false |long	|起始时间 |
|endTime |false	|long |截止时间 |
|limit |false |int |返回數據最大500條. 默認返回100條 |
|orderType |false |int |订单类型 1. 申购 2. 赎回 |
|serialNo |false |string |序列号 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |杠杆代币资产简称 |
|orderId |string |订单ID |
|orderType |string |订单类型 1. 申购 2. 赎回 |
|orderTime |string |下单时间 |
|excTime |string |订单状态最后一次更新时间 |
|orderStatus |string |订单状态：1. 已成交。2. 等待中。3. 失败 |
|fee |string |交易手续费 |
|amount	|string	|下单的杠杆代币资产数量 |
|value |string |成交价值 |
|valueCoin |string |计价币种 |
|serialNo |string |序列号 |
