Query a list of instruments of online trading pair.
> **Cover: Spot / USDT Perpetual / USDC Perpetual / Inverse Perpetual / Inverse Future / Option**

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP Request
GET `/v5/market/instruments-info`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |<b>true</b> |string |Product type. `spot`,`linear`,`inverse`,`option`|
|symbol |false |string |Symbol name|
|baseCoin |false |string |Base coin. for `option` ONLY.|
|limit |false |integer |Limit for data size per page. [1,1000]. default: `500`|
|cursor |false |integer |Cursor. Used to pagination|
