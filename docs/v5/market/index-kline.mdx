Query the index price kline data. This endpoint can retrieve the latest ? candles. Charts are returned in groups based on the requested interval.
> **Cover: USDT Perpetual / USDC Perpetual / Inverse Perpetual / Inverse Future**

### HTTP Request
GET `/v5/market/index-price-kline`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------|
|[category](../enum#category) |<b>true</b> |string |Product type. `linear`,`inverse`|
|symbol |<b>true</b> |string |Symbol name|
|[interval](../enum#interval) |<b>true</b> |string |Kline interval. `1`,`3`,`5`,`15`,`30`,`60`,`120`,`240`,`360`,`720`,`D`,`M`,`W`|
|start |false |integer |The start timestamp (ms)|
|end |false |integer |The end timestamp (ms)|
|limit |false |integer |Limit for data size per page. [1,200]. default: `200`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|symbol |string |Symbol name |
|list |array |<li>An string array of individual candle</li><li>Sort in reverse by `startTime`</li>|
|> list[0]: startTime |string |Start time of the candle (ms) |
|> list[1]: openPrice |string |Open price |
|> list[2]: highPrice |string |Highest price |
|> list[3]: lowPrice |string |Lowest price |
|> list[4]: closePrice |string |Close price. _To be last trade price when the candle is not closed_|
