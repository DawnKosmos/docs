Query the latest price snapshot, best bid/ask price, and trading volume in the last 24 hours.
> **Cover: Spot / USDT Perpetual / USDC Perpetual / Inverse Perpetual / Inverse Future / Option**

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP Request
GET `/v5/market/tickers`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `spot`,`linear`,`inverse`,`option` |
|symbol |false |string |Symbol name |
|baseCoin |false |string |Base coin. `option` valid ONLY, either pass `baseCoin` or `symbol`|
|expDate |false |string |Expiry date. e.g., 25DEC22. for `option` ONLY|
