Query option historical volatility
> **Cover: Option**

:::info
* The data is in hourly.
* If both `startTime` and `endTime` are not specified, it will return the recent 1-hour data by default.
* `startTime` and `endTime` is a pair of params. Either both are passed or not passed.
* This endpoint supports to query last 2 years data, but make sure [`endTime` - `startTime`] <= 30 days.
:::

### HTTP Request
GET `/v5/market/historical-volatility`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Product type. `option` |
|baseCoin |false |string |Base coin. default: return BTC data|
|[period](../enum#period) |false |integer |Period|
|startTime |false |integer |The start timestamp (ms) |
|endTime |false |integer |The end timestamp (ms) |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Product type |
|list |array |Object |
|> period |integer |Period |
|> value |string |Volatility |
|> time |string |Timestamp (ms) |

### Request Example

```http
GET /v5/market/historical-volatility?category=option&baseCoin=ETH&period=30 HTTP/1.1
Host: api-testnet.bybit.com
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "SUCCESS",
    "category": "option",
    "result": [
        {
            "period": 30,
            "value": "0.45024716",
            "time": "1672052400000"
        }
    ]
}
```
