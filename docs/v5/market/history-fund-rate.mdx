#### HTTP Request
GET `/v5/market/history-funding-rate`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |false |string |Product type. `linear`,`inverse`|
|symbol |<b>true</b> |string |Symbol name|
|start |false |number |The start timestamp (ms)|
|end |false |number |The end timestamp (ms)|
|limit |false |number |Limit for data size per page. [1,200]. default: `200`|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Products category. Keeps empty string `""` when `category` is not passed |
|list |array |Object |
|> symbol |string |Symbol name |
|> fundingRate |string |Funding rate |
|> fundingRateTimestamp |string |Funding rate timestamp(ms) |
