Query asset information

:::info
For now, it can query `SPOT` ONLY.
:::

### HTTP Request
GET `/v5/asset/transfer/asset-info-query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](../enum#accounttype) |false |string |Account type. `Spot` ONLY |
|[coin](../enum#accounttype) |false |string |To account type |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|spot |Object| |
|> status |string |account status. `ACCOUNT_STATUS_NORMAL`,`ACCOUNT_STATUS_UNSPECIFIED` |
|> assets |array|Object |
|>> coin |string|Coin |
|>> frozen |string|Freeze amount |
|>> free |string|Free balance |
|>> withdraw |string|Amount in withdrawing |
