Query asset information

:::info
For now, it can query `SPOT` ONLY.
:::

### HTTP Request
GET `/v5/asset/transfer/query-asset-info`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](../enum#accounttype) |**true** |string |Account type. `Spot` ONLY |
|coin |false |string |Coin name |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|spot |Object| |
|> status |integer |account status. `1`: normal, `0`: banned |
|> assets |array|Object |
|>> coin |string|Coin |
|>> frozen |string|Freeze amount |
|>> free |string|Free balance |
|>> withdraw |string|Amount in withdrawing |
