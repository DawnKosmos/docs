Withdraw asset from **SPOT** account. You can make an internal transfer if target wallet address is from bybit.

:::caution
Make sure you have added wallet address <a href="https://www.bybit.com/user/assets/money-address">here</a>
:::

### HTTP Request
POST `/v5/asset/withdraw/create`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin |**true** |string |Coin |
|chain |**true** |string |Chain |
|address |**true** |string |Wallet address |
|tag |false |string |Tag. **required** if tag existed in the wallet address list |
|amount |**true** |string |Withdraw amount |
|timestamp |**true** |integer |Current timestamp (ms). Used for preventing from withdraw replay |
|forceChain |false |integer |Force to withdraw on chain or not<li>`0`: If the address is parsed out to be internal address, then internal transfer (default)</li><li>`1`: Force withdraw on chain</li> |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|id |string|Withdraw id |
