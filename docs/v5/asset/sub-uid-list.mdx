Query sub uid list

:::caution
The API key of user id must own one of permissions will be allowed to call this API endpoint.
* SUB UID: "Account Transfer"
* MASTER UID: "Account Transfer", "Subaccount Transfer", "Withdrawal"
:::

### HTTP Request
GET `/v5/asset/transfer/query-sub-member-list`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subMemberIds |array&lt;string&gt; |All sub uids under the main uid |
|transferableSubMemberIds |array&lt;string&gt; |All sub uid that enabled universal transfer |
