Query the balance of a specific coin in a specific [account type](../enum#accounttype)

:::caution
Can query by main uid api key ONLY
:::

### HTTP Request
GET `/v5/asset/transfer/sub-member-list-query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|memberId |false |string |UID. **required** when query sub uid account type balance |
|[accountType](../enum#accounttype) |**true** |string |Account type |
|coin |**true** |string |Coin |
|withBonus |false |string |`0`: not query bonus. `1`: query bonus  |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|accountType |string |Account type |
|bizType |integer |Biz type. 确认下枚举值和含义 |
|accountId |string |Account id |
|memberId |string |Uid |
|balance |Object | |
|> coin |string |Coin |
|> walletBalance |string |Wallet balance |
|> transferBalance |string |Transferable balance |
|> bonus |string |bonus |
