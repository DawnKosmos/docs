Transfer between sub-sub or main-sub. Please make sure your sub uid [enables universal transfer](enable-unitransfer-subuid#) in advance.

:::caution
Can query by main uid api key ONLY
:::

### HTTP Request
GET `/v5/asset/transfer/universal-transfer`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|transferId |**true** |string |UUID. Please manually generate a UUID |
|coin |**true** |string |Coin |
|amount |**true** |string |Amount |
|fromMemberId |**true** |integer |From UID |
|toMemberId |**true** |integer |TO UID |
|[fromAccountType](../enum#accounttype) |**true** |string |From account type |
|[toAccountType](../enum#accounttype) |**true** |string |To account type |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|transferId |string|UUID |
