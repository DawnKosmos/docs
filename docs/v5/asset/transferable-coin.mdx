Query transferable coin list between each [account type](../enum#accounttype)

### HTTP Request
GET `/v5/asset/transfer/query-transfer-coin-list`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[fromAccountType](../enum#accounttype) |**true** |string |From account type |
|[toAccountType](../enum#accounttype) |**true** |string |To account type |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list |array|each string coin in the array |
