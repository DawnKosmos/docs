Cancel the withdrawal

### HTTP Request
GET `v5/asset/withdraw/cancel`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|id |**true** |string |Withdraw id |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status |integer|`0`: fail. `1`: success |
