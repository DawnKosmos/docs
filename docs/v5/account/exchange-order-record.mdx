Get the currency exchange record of the current unified margin account, and do not support pagination. Only the last 500 datas can be queried. Sort in descending order of creation time.


#### HTTP Request
GET `/v5/asset/exchange/order-record`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fromCoin |false |string |Exchange currency, such as BTC|
|toCoin |false |string |Currency exchanged, such as USDT|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|totalCount |int |Sum of data |
|list > fromCoin |string |Exchange currency|
|list > fromAmount |string |exchange amount |
|list > toCoin |string |Currency exchanged|
|list > toAmount |string |Currency exchanged amount |
|list > exchangeRate |string |Exchange Rate |
|list > createdTime |string |Created Time, timestamps in milliseconds|
|list > exchangeTxId |string |exchange TxId|
