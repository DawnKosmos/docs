Upgrade Unified Account

:::note important
Banned / OTC loan / Net asset unsatisfying / Express path users cannot upgrade the account to Unified Account for now.
:::

### HTTP Request
POST `/v5/account/upgrade-to-uta`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|unifiedUpdateStatus |string | Upgrade status. `1`: Processing, `2`: Success, `3`: Failed, `4`: Rejected |
|unifiedUpdateMsg |array |Object |
|> msg |string |Error message returned |
