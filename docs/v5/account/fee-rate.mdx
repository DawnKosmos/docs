获取当前用户的交易手续费

:::caution
限频，每分钟只能查询 5 次
:::


#### HTTP Request
GET `/v5/account/fee-rate`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |交易产品类型，现货：spot，正向合约：linear；反向合约：inverse；期权：option。|
|symbol |false |string |合约名称|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |交易产品类型，现货：spot，正向合约：linear；反向合约：inverse；期权：option。|
|list > symbol |string |交易产品名称|
|list > takerFeeRate |string |吃单手续费率|
|list > makerFeeRate |string |挂单手续费率|
