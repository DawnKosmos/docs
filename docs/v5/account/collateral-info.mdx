Get the collateral information of the current unified margin account, including loan interest rate, loanable amount, collateral conversion rate, whether it can be mortgaged as margin, etc.


#### HTTP Request
GET `/v5/account/collateral-info`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |Asset currency of all current collateral|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list > currency |string |Currency of all current collateral|
|list > hourlyBorrowRate |string |Hourly borrow rate|
|list > maxBorrowAmount |string |Max borrow amount|
|list > freeBorrowAmount |string |Free borrow amount|
|list > borrowAmount |string |Borrow amount|
|list > availableToBorrow |string |Available amount to borrow|
|list > borrowable |bool |Whether currency can be borrowed: true yes; false no|
|list > marginCollateral |bool |Whether it can be used as a margin collateral currency: true 是; false 否|
|list > collateralStatus |string |Whether it can be used as collateral status：1 是；0 否|
|list > collateralRatio |string |Collateral ratio|
