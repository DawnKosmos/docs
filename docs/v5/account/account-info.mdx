Query the configuration under the account. The first version is to query the margin mode. Supports RM and PM.

:::caution
In the future, the endpoint will add the following configuration queries: whether it is in a tradable status, whether it can be transferred, and other account configurations.
:::


#### HTTP Request
GET `/v5/account/info`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|dcpStatus |string |ON/OFF  Query the current time window configuration and DCP switch status. Frequency limit: 10 times/second|
|timeWindow |string |Disconnection timing window time|
|unifiedMarginStatus |string |1: Regular non unified margin account. 2: The unified margin account has been upgraded, only the futures and options. 3: Upgraded unified margin account, futures,options and spot|
|marginMode |string |REGULAR_MARGIN, PORTFOLIO_MARGIN|
|updateTime |string |update time, timestamp in milliseconds|
