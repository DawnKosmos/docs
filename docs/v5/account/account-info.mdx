Query the configuration under the account.

:::note
Query the margin mode and the upgraded status of account
:::

### HTTP Request
GET `/v5/account/info`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|[unifiedMarginStatus](../enum#unifiedmarginstatus) |integer |Unified margin account status |
|marginMode |string |`REGULAR_MARGIN`, `PORTFOLIO_MARGIN`|
|updatedTime |string |Account data updated timestamp (ms)|
