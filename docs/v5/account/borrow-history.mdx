Get interest records, sorted in reverse order of creation time.

#### HTTP Request
GET `/v5/account/borrow-history`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |USDC、USDT、BTC、ETH|
|startTime |false |int |Start time. timestamp in milliseconds|
|endTime |false |int |End time. timestamp in milliseconds|
|limit |false |integer |size per page, Maximum 50. The default is 20 items per page|
|cursor	|false |string |cursor for pagination, please use the returned cursor, use the returned cursor when generating signature. Please use urlencode when sending the request|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list > currency |string |USDC、USDT、BTC、ETH|
|list > createdTime |int64 |created time, timestamp in milliseconds|
|list > borrowCost |string |Interest|
|list > hourlyBorrowRate |string |Hourly Borrow Rate
|list > InterestBearingBorrowSize |string |Interest Bearing Borrow Size
|list > borrowSize |string |borrowe size
|list > costExemption |string |cost exemption
