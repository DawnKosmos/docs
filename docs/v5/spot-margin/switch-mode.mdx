Turn on / off spot margin trade.

:::caution
You account needs to enable spot margin first, i.e., you have finished quiz on web / app.
:::

### HTTP Request
POST `/v5/spot-margin-trade/switch-mode`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|spotMarginMode |**true** |integer |`1`: on，`0`: off |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|spotMarginMode |string |Spot margin status. `1`: on，`0`: off|
