Set the user's maximum leverage in spot cross margin

:::caution
Your account needs to enable spot margin first; i.e., you must have finished the quiz on web / app.
:::

### HTTP Request
GET `/v5/spot-margin-trade/set-leverage`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|leverage |**true**| integer |Leverage. [`2`, `5`].|

### Response Parameters
None

### Request Example

```http
POST /v5/spot-margin-trade/set-leverage HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1672299806626
X-BAPI-RECV-WINDOW: 5000
Content-Type: application/json

{
"leverage": 4
}
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {},
    "retExtInfo": {},
    "time": 1672710944282
}
```
