:::tip
Set the risk limit of the position. You could get risk limit information of each symbol [here](../../rest-public/risk-limit#).
:::

### HTTP Request
POST `/contract/v3/private/position/set-risk-limit`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol |**true** |string |Symbol name|
|riskId |**true** |integer |Risk limit id|
|positionIdx |**true** |integer |Used to identify positions in different position modes<li>`0`: one-way mode</li><li>`1`: hedge-mode Buy side</li><li>`2`: hedge-mode Sell side</li>|

### Response Parameters
None
