Set the leverage.

### HTTP Request
POST `/contract/v3/private/position/set-leverage`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|symbol |**true** |string |Symbol name|
|buyLeverage |**true** |string |(0, max leverage of corresponding [risk limit](../public/risk-limit#)] |
|sellLeverage |**true** |string |(0, max leverage of corresponding risk limit]|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|retCode |integer |Success/Error code |
|retMsg |string |Success/Error message |
|result |Object | |
|retExtInfo |Object | |
|time |integer |Timestamp |
