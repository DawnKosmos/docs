Get USDC account margin mode

### HTTP Request
GET `/contract/v3/private/account/info`

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|------ |
|marginMode |string |Margin mode. `REGULAR_MAEGIN`,`PORTFOLIO_MARGIN` |
|updateTime |string |Account config update timestamp (ms) |

### Request Example

```http
GET /contract/v3/private/account/info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1670941248049
X-BAPI-RECV-WINDOW: 5000
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "Success.",
    "result": {
        "marginMode": "REGULAR_MARGIN",
        "updateTime": "1669888247000"
    }
}
```
