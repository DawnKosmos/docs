Default is the regular margin mode. This mode is only valid for USDC Perp and USDC Option. Portfolio Margin does not support USDT-settled contracts.

### HTTP Request
POST `/unified/v3/private/account/setMarginMode`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|setMarginMode |**true** |string |`REGULAR_MARGIN`, `PORTFOLIO_MARGIN`|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|reasons |array|Empty array if the request is submitted successfully. Otherwise, there are failure Object |
