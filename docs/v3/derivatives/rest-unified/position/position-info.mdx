Get real-time position data

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP Request
GET `/unified/v3/private/position/list`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../../enum#category) |**true** |string |Product type. `linear`, `option` |
|symbol |false |string |Symbol name |
|baseCoin |false |string |Base coin. <li>`category`=option, then **BTC** records returned by default</li> |
|direction |false |string |Page direction. `prev`, `next`. default: `next`|
|limit |false |string |Limit for data size per page. [1,50]. default: `20`|
|cursor |false |string |Cursor. Used to pagination |
