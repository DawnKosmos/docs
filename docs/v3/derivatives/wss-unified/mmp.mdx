Subscribe to the MMP event stream, `option` only.

:::info
There is feed only when MMP is triggered.
:::

**Topic:**<br/>
`user.mmpEvent`

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|triggerStatus |string |MMP trigger status |
|triggerTimestamp |string |MMP trigger timestamp (ms) |
|frozenPeriod |string |Forzen period (ms)|
|baseCoin |string |Base coin|
