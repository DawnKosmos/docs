Query Master Acct Deposit Address Information

### HTTP Request
GET `/asset/v3/private/deposit/address/query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[coin](../enum#coin) |false |string |coin name|
|chainType |false |string |Chain name. Return all if not passed.<ul><li>pass chain name, e.g. `ETH` instead of `ERC20`</li></ul>|


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result|Object|result object|
|[coin](../enum#coin)|string |Coin name |
|chains|Array|Object|
|chainType|string|chain type|
|addressDeposit|string|deposit address|
|tagDeposit|string|deposit tag|
|chain|string|chain name|

### Request Example

```http
GET /asset/v3/private/deposit/address/query?coin=USDT&chainType=ETH HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: xxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxx
X-BAPI-TIMESTAMP: 1671174181320
X-BAPI-RECV-WINDOW: 50000
Content-Type: application/x-www-form-urlencoded
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
        "coin": "USDT",
        "chains": [
            {
                "chainType": "ERC20",

            }
        ]
    },

    "time": 1671174181514
}
```
