Create a new sub user id. Use `master user's api key` ONLY.

### HTTP Request
POST `/user/v3/private/create-sub-api`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|username |<b>true</b> |string |Give a username of the new sub user id. <li>6-16 characters, must include both numbers and letters.</li><li>cannot be the same as the exist or deleted one.</li> |
|memberType |<b>true</b> |int |`1`: normal sub account, `6`: custodial sub account|
|switch |false |int |<li> `0`: turn off quick login (default)</li><li>  `1`: turn on quick login.</li>|
|note |false |string |Set a remark |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|uid |int |Sub user Id |
|username |string |Give a username of the new sub user id. <li>6-16 characters, must include both numbers and letters.</li><li>cannot be the same as the exist or deleted one.</li> | |
|memberType |<b>true</b> |int |`1`: normal sub account, `6`: custodial sub account|
|switch |false |int |<li> `0`: turn off quick login (default)</li><li>  `1`: turn on quick login.</li>|
|note |false |string |Set a remark |
