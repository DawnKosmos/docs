Get all sub uid of master account. Use `master user's api` key ONLY.

### HTTP Request
POST `/user/v3/private/query-sub-members`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subMembers |array |Object |
|uid |string |Sub user Id |
|username |string |Username|
|memberType |string |`1`: normal sub account, `6`: custodial sub account |
|status |string |The status of the user account<li>`1`: normal</li><li>`2`: login banned</li><li>`4`: frozen </li>|
|remark |string |The remark |
