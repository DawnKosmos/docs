Query Subaccount List

### HTTP Request
GET `/asset/v3/private/transfer/sub-member/list/query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |


### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|subMemberIds |`integer array` |List of all subaccounts|
|transferableSubMemberIds|`string array`	|List of subaccounts that are universal transfer-enabled|
