Get the Asset Info

### HTTP Request
GET `/asset/v3/private/transfer/asset-info/query`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[accountType](../enum#accounttype)|false |string |Account type, `SPOT` in default. For now, only `SPOT` is available. |
|[coin](../enum#coin)|false |string |Currency type|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status |string |<li>`ACCOUNT_STATUS_NORMAL` -- NORMAL</li> <li>`ACCOUNT_STATUS_UNSPECIFIED` -- ABNORMAL</li>|
|assets|list|Object|
|[>coin](../enum#coin)|string |Currency type |
|>frozen |string |Freeze amount|
|>free |string |Free balance |
|>withdraw |string |Amount in withdrawing |
