Enable Universal Transfer

:::tip
A universal transfer enables you to transfer funds directly between two subaccounts (rather than sub -> master -> sub) and any [accountType](../enum#accounttype) (CONTRACT, SPOT, etc).
:::

Use this endpoint to enable subaccounts to take part in a universal transfer. It is a one-time switch which, once thrown, enables a subaccount permanently. If not set, your subaccounts cannot use universal tranfers.

### HTTP Request
POST `/asset/v3/private/transfer/transfer-sub-member-save`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|subMemberIds |false|string |A comma-separated list of subaccount UIDs, for example "123,45,14,26,46" |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
