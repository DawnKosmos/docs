Get User Register Date

### HTTP Request
POST `/fht/compliance/tax/v3/private/registertime`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|registerTime|string|UNIX Time. This is in milleseconds. Date that the specific user registers on the platform|
