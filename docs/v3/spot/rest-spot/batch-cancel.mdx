Get the spec of symbol information

#### HTTP Request
POST <code>/spot/v3/private/cancel-orders</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
| symbol                | <b>true</b>         | string          | Name of the trading pair      |
| side                  | false               | string          | Side. <code>Buy</code>, <code>Sell</code> |
| orderTypes            | false               | string          | Order type. <code>LIMIT</code> in default. It allows multiple types, separated by comma, e.a LIMIT,LIMIT_MAKER |
| orderCategory        | false               | int             | Order category. <code>0</code>：normal order by default; <code>1</code>：<a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=us_EN&id=000001736">TP/SL order</a>, <b>Required</b> for TP/SL order.    |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| success              | string          | Batch cancel successfully or not. <code>0</code>：fail, <code>1</code>：success     |
