#### HTTP Request
GET <code>/spot/v3/private/cross-margin-account</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|acctBalanceSum|string|Total Equity (BTC)|
|debtBalanceSum|string|Total Liability (BTC)|
|loanAccountList|array|Object|
|> free|string|Available|
|> interest|string|Interest Left|
|> loan|string|Total Liability|
|> locked|string|Locked|
|> tokenId|string|Token Id|
|> total|string|Total|
|riskRate|string|Risk Rate|
|status|int|Status (1. Normal 2. Withdrawal/Transfer Restricted 3. Liquidation Alert Triggered 4. Liquidated 5. Negative Equity)|
