### HTTP Request
GET <code>/spot/v3/private/cross-margin-repay</code>

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|<a href="#currency-currency-coin">coin</a>|<b>true</b>|string|currency|
|qty|<b>true</b>|string|Repayment Amount|

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|repayId|string|Repay id|

### Request Example

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/cross-margin-repay' \
--header 'Content-Type: application/json' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662618298354' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXX' \
--data-raw '{
    "coin": "USDT",
    "qty": "150"
}'
```

### Response Example

```json
{
    "retCode": 0,
    "retMsg": "success",
    "result": {
        "repayId": "12128"
    },
    "retExtInfo": null,
    "time": 1662618298452
}
```
