#### HTTP Request
POST <code>/spot/v3/private/cross-margin-loan</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|<a href="#currency-currency-coin">coin</a>|<b>true</b>|string|currency|
|qty|<b>true</b>|string|Amount To Borrow|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|transactId|string|Loan order id|
