#### HTTP Request
GET <code>/spot/v3/public/quote/trades</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
| symbol | **true** | string | Name of the trading pair |
| limit | false | integer | Default value is 60, max 60 ｜

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| price | string | Order price |
| time | long | Current timestamp, unit in millisecond |
| qty | string | Order quantity |
| isBuyerMaker | integer | `0`：buy order, `1`：sell order |

### Request Example
