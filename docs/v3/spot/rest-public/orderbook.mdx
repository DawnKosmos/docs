#### HTTP Request
GET <code>/spot/v3/public/quote/depth</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| time | long |  Current time, unit in millisecond |
| bids | array | Bid price and quantity, with best bid prices ranked from top to bottom |
| asks | array | Ask price and quantity, with best ask prices ranked from top to bottom |

### Request Example

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/depth?symbol=BTCUSDT&limit=1'
```
