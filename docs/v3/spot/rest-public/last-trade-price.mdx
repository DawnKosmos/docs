#### HTTP Request
GET <code>/spot/v3/public/quote/ticker/price</code>


#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|<a href="#symbol-symbol">symbol</a> |false|string|Name of the trading pair|


#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| <a href="#symbol-symbol">symbol</a> | string | Name of the trading pair |
| <a href="#price-price">price</a> | string |  Last traded price |

### Request Example

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/ticker/price?symbol=BTCUSDT'
```

</TabItem>
</Tabs>
