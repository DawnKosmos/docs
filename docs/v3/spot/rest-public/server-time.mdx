### HTTP Request
GET <code>/spot/v3/public/server-time</code>

### Request Parameters
None

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| serverTime | string | Bybit server timestamp (ms)  |

### Request Example

```http
curl https://api-testnet.bybit.com/spot/v3/public/server-time
```

### Response Example

```json
{
  "retCode": 0,
  "retMsg": "OK",
  "result": {
    "serverTime": "1670521545784"
  },
  "retExtInfo": {},
  "time": 1670521545785
}
```
