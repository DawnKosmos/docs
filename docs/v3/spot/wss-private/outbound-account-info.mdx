<p>This topic pushes account information</p>
<p><b>Push frequency</b>: real-time</p>
<p><b>Topic</b>: <code>outboundAccountInfo</code></p>

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| e | string | Event type |
| E | string | Timestamp |
| T | boolean | Allow trade |
| W | boolean | Allow withdraw |
| D | boolean | Allow deposit|
| B | list | Wallet balance change |
| a | string | coin name |
| f | string | Available balance |
| l | string | Reserved for orders |
