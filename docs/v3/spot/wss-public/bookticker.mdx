
<p>Best bid price and best ask price</p>
<p><b>Push frequency</b>: 100ms</p>
<p><b>Topic</b>: <code>bookticker.BTCUSDT</code></p>


#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| s | string | Trading pair |
| bp | string | Best bid price |
| bq | string | Bid quantity |
| ap | string | Best ask price|
| aq | boolean | Ask quantity |
| t | long | timestamp(ms). The time that system generates the data. |
| ts | long | The time that message is sent out |
