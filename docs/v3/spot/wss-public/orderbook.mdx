<p>Market depth data for a trading pair: </p>
<ul>
  <li>Snapshot depth: <b>40</b> each for asks and bids.</li>
  <li>Events trigger order book version change:
    <ul>
      <li>order enters order book</li>
      <li>order leaves order book</li>
      <li>order quantity changes</li>
      <li>order filled</li>
    </ul>
  </li>
</ul>
<p>Pushes snapshot data only, please ignore type=<code>delta</code> in the message</p>
<p><b>Push frequency</b>: 100ms</p>
<p><b>Topic</b>: <code>orderbook.40.BTCUSDT</code></p>

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| t | long | timestamp(ms). The time that system generates the data. |
| s | string | Trading pair |
| b | list | Bid price, quantity |
| a | list | Ask price, quantity |
| ts | long | The time that message is sent out |

### Request Example

<Tabs>
<TabItem value="console" label="JSON subscription message">
