:::info
Do not use the duplicate **orderLinkId** in normal order & TP/SL order
:::
#### HTTP Request
POST <code>/spot/v3/private/order</code>

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
| symbol               | <b>true</b>         | string          | Name of the trading pair              |
| orderQty             | <b>true</b>         | string          | |       |
| side                 | <b>true</b>         | string          | Side. <code>Buy</code>, <code>Sell</code>     |
| orderType            | <b>true</b>         | string          | Order type           |
| <a href="#time-in-force-time_in_force">timeInForce</a>| false               | string          | <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001044">Time in force</a> |
| orderPrice           | false               | string          | Order price. When the type field is MARKET, the price field is optional. When the type field is LIMIT or LIMIT_MAKER, the price field is required      |
| orderLinkId          | false               | string          | User-generated order ID         |
| orderCategory        | false               | int             | Order category. <code>0</code>：normal order by default; <code>1</code>：<a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=us_EN&id=000001736">TP/SL order</a>, <b>Required</b> for TP/SL order.    |
| triggerPrice         | false               | string          | Trigger price     |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
| orderId              | string          | Order ID             |
| orderLinkId          | string          | User-generated order ID         |
| symbol               | string          | Name of the trading pair              |
| createTime           | string          | Order Creation Time        |
| orderPrice           | string          |  Last traded price               |
| orderQty             | string          | Order quantity              |
| orderType            | string          | Order type           |
| side                 | string          | Side. <code>Buy</code>, <code>Sell</code>     |
| status               | string          | Order status              |
| timeInForce          | string          | <a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=en_US&id=000001044">Time in force</a> |
| accountId            | string          | Account ID           |
| execQty              | string          | Please ignore             |
| orderCategory        | int             | Order category. <code>0</code>：normal order by default; <code>1</code>：<a href="https://www.bybit.com/en-US/help-center/bybitHC_Article?language=us_EN&id=000001736">TP/SL order</a>, <b>Required</b> for TP/SL order.    |
| triggerPrice         | string          | Trigger price     |
