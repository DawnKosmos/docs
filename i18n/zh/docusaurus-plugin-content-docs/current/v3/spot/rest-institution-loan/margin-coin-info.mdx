#### HTTP 請求
GET <code>/spot/v3/public/margin-ensure-tokens</code>

#### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|productId | false | string	| 產品ID. 若不傳，則返回所有產品的保證金幣種信息. 現貨返回折算率大於0的幣種. |

#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|marginToken |array	| Object |
|> productId |string | 產品ID |
|> spotToken |array	| 現貨保證金幣種信息 |
|>> token |string | 保證金幣種 |
|>> convertRatio |string | 保證金幣種折算率 |
|> contractToken |array	| 合約保證金幣種信息n |
|>> token |string | 保證金幣種 |
|>> convertRatio |string | 保證金幣種折算率 |

### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/public/margin-ensure-tokens?productId=70'
```

</TabItem>
</Tabs>
