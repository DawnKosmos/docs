#### HTTP 請求
GET <code>/spot/v3/public/margin-product-infos</code>

#### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|productId | false | string	| 產品ID. 若不傳，則返回所有產品數據 |

#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|marginProductInfo	array	Object
|> productId | string | 產品ID |
|> leverage	| string | 槓桿 |
|> supportSpot | integer | 是否支持現貨. 0:false; 1:true |
|> supportContract | integer | 是否支持合約. 0:false; 1:true |
|> withdrawLine	| string | 限制提幣線 |
|> transferLine	| string | 限制劃轉線 |
|> spotBuyLine | string	| 限制現貨買入線 |
|> spotSellLine	| string | 限制現貨賣出線 |
|> contractOpenLine	| string | 限制合約開倉線 |
|> liquidationLine | string	| 強平線 |
|> stopLiquidationLine | string	| 停止強平線 |
|> contractLeverage	| string | 允許合約開倉倍數 |
|> transferRatio | string | 借貸資金劃轉比例 |
|> spotSymbols | array | 現貨交易對白名單 |
|> contractSymbols | array | 合約交易對白名單 |


### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/margin-product-infos?productId=70'
```

</TabItem>
</Tabs>
