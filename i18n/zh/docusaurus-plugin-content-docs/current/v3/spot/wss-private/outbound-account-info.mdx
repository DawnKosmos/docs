
<p>該訂閱用於推送帳戶信息</p>
<p><b>推送頻率</b>: 實時</p>
<p><b>Topic</b>: <code>outboundAccountInfo</code></p>


#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| e | string | 事件類型 |
| E | string | 時間戳 |
| T | boolean | 允許交易 |
| W | boolean | 允許提現 |
| D | boolean | 允許充值 |
| B | list | Wallet balance change |
| a | string | 幣名 |
| f | string | 可用數量 |
| l | string | 凍結余額 |
