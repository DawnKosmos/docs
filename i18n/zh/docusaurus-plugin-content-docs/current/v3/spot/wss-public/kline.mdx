
<p>訂閱Kline需要提供間隔參數，最短為分鐘線，最長為月線。支持以下間隔: m -> 分鐘; h -> 小時; d -> 天; w -> 周; M -> 月</p>
<p><b>推送頻率</b>: 實時</p>
<p><b>Topic</b>: <code>kline.1m.BTCUSDT</code></p>


#### 返回參數
| 參數 | 參數 | 說明|
|:----- |:-----|----- |
|t | long | 開始時間戳（毫秒) |
|s | string | 幣對 |
|c | string | 收盤價 |
|h | string | 最高價 |
|l | string | 最低價 |
|o | string | 開盤價 |
|v | string | 成交量 |
| ts | long | 消息被推送出來的時間戳 |
