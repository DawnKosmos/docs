獲取Bybit服務器時間


#### HTTP 請求
GET <code>/spot/v3/public/server-time</code>

#### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |

#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| serverTime | string | 服務器時間 |

### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
curl https://api-testnet.bybit.com/spot/v3/public/server-time
```

</TabItem>
</Tabs>
