Get open interest of each symbol in Bybit

#### HTTP Request
GET `/derivatives/v3/public/open-interest`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |false |string |Products category. `future` ONLY |
|symbol |<b>true</b> |string |Symbol name|
|interval |<b>true</b> |string |Interval. `5min` `15min` `30min` `1h` `4h` `1d` |
|startTime |false |string |The start timestamp (ms)|
|endTime |false |string |The end timestamp (ms)|
|limit |false |string |Limit for data size per page. [1,200]. default: `50`|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|symbol |string |Symbol name |
|category |string |Products category. Keeps empty string `""` when `category` is not passed |
|list |array |Object |
|> openInterest |string |Open interest |
|> timestamp |string |The timestamp (ms) |
