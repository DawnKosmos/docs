Set the leverage.

#### HTTP Request
POST `/unified/v3/private/position/set-leverage`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |Products category. `linear` ONLY |
|symbol |**true** |string |Symbol name|
|buyLeverage |**true** |string |(0, allowed max leverage of corresponding risk limit] |
|sellLeverage |**true** |string |(0, allowed max leverage of corresponding risk limit]|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|retCode |integer |Success/Error code |
|retMsg |string |Success/Error message |
|result |Object | |
|time |integer |Timestamp |
