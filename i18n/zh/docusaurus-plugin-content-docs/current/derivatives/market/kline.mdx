Get kline data.

#### HTTP Request
GET `/derivatives/v3/public/kline`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |false |string |Products category. `future` ONLY |
|symbol |<b>true</b> |string |Symbol name|
|interval |<b>true</b> |string |Kline interval. `1` `3` `5` `15` `30` `60` `120` `240` `360` `720` `D` `M` `W`  |
|start |<b>true</b> |number |The start timestamp (ms)|
|end |<b>true</b> |number |The end timestamp (ms)|
|limit |false |number |Limit for data size per page. [1,200]. default: `200`|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|symbol |string |Symbol name |
|category |string |Products category. Keeps empty string `""` when `category` is not passed |
|list |array |<li>An string array of individual candle</li><li>Sort in reverse by `start`</li><li>The default collation within the array is `start`, `open`, `high`, `low`, `close`, `volume`, `turnover`</li> |
