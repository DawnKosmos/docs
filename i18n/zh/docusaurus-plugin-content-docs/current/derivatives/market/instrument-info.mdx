獲取可交易的衍生品信息列表

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

#### HTTP 請求
GET `/derivatives/v3/public/tickers`

#### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|category |false |string |Products category. `future`, `option`. future data returned by default if both `category` and `symbol` are not passed |
|symbol |false |string |Symbol name. <li>query an option symbol, `category` is **required**</li>|
|baseCoin |false |string |Base coin. <li>Valid when `category`=option</li><li>BTC by default if not passed</li>|
|limit |false |number |Limit for data size per page. [1,1000]. default: 500|
|cursor |false |string |Cursor. Used to pagination|
