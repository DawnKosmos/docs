申贖記錄查詢

:::caution
API權限:讀寫權限
:::


#### HTTP Request
POST `/v5/spot-lever-token/order-record`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |**true** |string |LT資產名稱，例如BTC3L |
|orderId |false	|string	|訂單ID |
|startTime |false |long	|起始時間 |
|endTime |false	|long |截止時間 |
|limit |false |int |返回數據最大500條. 默認返回100條 |
|orderType |false |int |訂單類型 1. 申購 2. 贖回 |
|serialNo |false |string |序列號 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |槓桿代幣資產簡稱 |
|orderId |string |訂單ID |
|orderType |string |訂單類型 1. 申購 2. 贖回 |
|orderTime |string |下單時間 |
|excTime |string |訂單狀態最後一次更新時間 |
|orderStatus |string |訂單狀態：1. 已成交。 2. 等待中。 3. 失敗 |
|fee |string |交易手續費 |
|amount	|string	|下單的槓桿代幣資產數量 |
|value |string |成交價值 |
|valueCoin |string |計價幣種 |
|serialNo |string |序列號 |
