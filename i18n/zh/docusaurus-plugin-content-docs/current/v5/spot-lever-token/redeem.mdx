赎回

:::caution
API权限:读写权限
:::


#### HTTP Request
POST `/v5/spot-lever-token/redeem`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |**true** |string |LT资产名称，例如BTC3L |
|ltQuantity |**ture** |string |下单金额 |
|serialNo |false |string |序列号 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |杠杆代币资产简称 |
|orderStatus |string |申购单状态：1. 已完成 2. 等待中 3. 失败 |
|quantity |string |下单的杠杆代币资产数量 |
|orderQuantity |string |实际成交的杠杆代币资产数量 |
|orderAmount |string |实际成交的金额 |
|id	|string	|交易单号 |
|valueCoin |string |计价币种 |
|serialNo |string |序列号 |
