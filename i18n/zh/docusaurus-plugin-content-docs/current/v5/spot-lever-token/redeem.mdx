贖回

:::caution
API權限:讀寫權限
:::


#### HTTP Request
POST `/v5/spot-lever-token/redeem`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |**true** |string |LT資產名稱，例如BTC3L |
|ltQuantity |**ture** |string |下單金額 |
|serialNo |false |string |序列號 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |槓桿代幣資產簡稱 |
|orderStatus |string |申購單狀態：1. 已完成 2. 等待中 3. 失敗 |
|quantity |string |下單的槓桿代幣資產數量 |
|orderQuantity |string |實際成交的槓桿代幣資產數量 |
|orderAmount |string |實際成交的金額 |
|id	|string	|交易單號 |
|valueCoin |string |計價幣種 |
|serialNo |string |序列號 |
