LT行情查询

:::caution
API权限：不需要APIKEY
:::


#### HTTP Request
GET `/v5/spot-lever-token/reference`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |false |string |LT资产名称，例如BTC3L|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |杠杆代币资产简称 |
|nav |string |净值 |
|navTime |string |净值更新时间戳UTC毫秒 |
|circulation |string |在二级市场流通的代币数量 |
|basket	|string	|篮子 |
|leverage |string |真實槓桿倍數 |
