LT行情查詢

:::caution
API權限：不需要APIKEY
:::


#### HTTP Request
GET `/v5/spot-lever-token/reference`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |false |string |LT資產名稱，例如BTC3L|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |槓桿代幣資產簡稱 |
|nav |string |淨值 |
|navTime |string |淨值更新時間戳UTC毫秒 |
|circulation |string |在二級市場流通的代幣數量 |
|basket	|string	|籃子 |
|leverage |string |真實槓桿倍數 |
