LT全量资产查询

:::caution
API权限：不需要APIKEY
:::


#### HTTP Request
GET `/v5/spot-lever-token/infos`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|ltCoin |false |string |LT资产名称，例如BTC3L|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|ltCoin |string |杠杆代币资产简称 |
|ltName |string |杠杆代币资产全称 |
|maxPurchase |string |单次最大申购金额 |
|minPurchase |string |单次最小申购金额 |
|maxPurchaseDaily |string |单个自然日最大申购金额 |
|maxRedeem |string |单次最大赎回数量 |
|minRedeem |string |单次最小赎回数量 |
|maxRedeemDaily |string |单个自然日最大赎回数量 |
|purchaseFeeRate |string |申购费率 |
|redeemFeeRate |string |赎回费率 |
|status |string |目前杠杆代币资产是否可以进行申购赎回,1. 申赎均正常 2. 仅可申购，不可赎回 3. 仅可赎回，不可申购 4. 申赎均暂停|
|fundFee |string |每天针对持有杠杆代币资产的用户收取的资金费用 |
|fundFeeTime |string |收取资金费用的时间 |
|manageFeeRate |string |管理费率|
|manageFeeTime |string |收取资金管理费的时间 |
|value |string |名义资产规模 |
|total |string |总上限|
|netValue |string |当前净值 |
