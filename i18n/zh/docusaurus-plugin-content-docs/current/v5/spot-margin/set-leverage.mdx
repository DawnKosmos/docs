全倉槓桿設置用戶最大槓桿倍數

:::caution
需要先開啟全倉槓桿，才能調整槓桿。
:::

### HTTP 請求
GET `/v5/spot-margin-trade/set-leverage`

### 請求參數
| 參數 | 是否必需 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|leverage |**true**| integer |槓桿倍數（2-5）|

### 響應參數
None
