獲取當前用戶的交易手續費

:::caution
限頻，每分鐘只能查詢 5 次
:::


#### HTTP Request
GET `/v5/account/fee-rate`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |**true** |string |交易產品類型，現貨：spot，正向合約：linear；反向合約：inverse；期權：option。|
|symbol |false |string |合約名稱|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |交易產品類型，現貨：spot，正向合約：linear；反向合約：inverse；期權：option。|
|list > symbol |string |交易產品名稱|
|list > takerFeeRate |string |吃單手續費率|
|list > makerFeeRate |string |掛單手續費率|
