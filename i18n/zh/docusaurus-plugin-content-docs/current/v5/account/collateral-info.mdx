獲取當前統一保證金賬戶的抵押品信息，包括借貸利率，可藉貸金額以及抵押品折算率，是否可抵押作為保證金等信息。

#### HTTP Request
GET `/v5/account/collateral-info`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |目前所有抵押品的資產幣種|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list > currency |string |目前所有抵押品的資產幣種|
|list > hourlyBorrowRate |string |每小時藉款利率|
|list > maxBorrowAmount |string |最大可藉貸額度|
|list > freeBorrowAmount |string |免息借貸額度|
|list > borrowAmount |string |已用借貸額度|
|list > availableToBorrow |string |用戶剩餘可藉額度|
|list > borrowable |bool |是否是可藉貸的借貸幣種 : true 是; false 否|
|list > marginCollateral |bool |是否可作為保證金抵押幣種: true 是; false 否|
|list > collateralStatus |string |是否能作為抵押品的狀態：1 是；0 否|
|list > collateralRatio |string |抵押品幣種折算為保證金的折算率|
