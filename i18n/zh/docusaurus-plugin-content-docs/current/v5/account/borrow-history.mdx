獲取利息記錄，按照創建時間倒敘排列。

#### HTTP Request
GET `/v5/account/borrow-history`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |USDC、USDT、BTC、ETH|
|startTime |false |int |開始時間|
|endTime |false |int |結束時間|
|limit |false |integer |每頁數量, 最大50. 默認每頁20條|
|cursor |false |string |翻頁標記,請使用返回的cursor,簽名時使用返回的原始數據，發送request時請進行urlencode|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list > currency |string |USDC、USDT、BTC、ETH|
|list > createdTime |int64 |時間戳（毫秒）|
|list > borrowCost |string |利息|
|list > hourlyBorrowRate |string |利率|
|list > InterestBearingBorrowSize |string |豁免利息額度|
|list > borrowSize |string |計息借款|
|list > costExemption |string |豁免計息成本|
