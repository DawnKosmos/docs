獲取當前統一保證金賬戶的幣幣兌換記錄，兌換記錄查詢（全量）, 不支持分頁。只能查詢最近 500 條。按照創建時間倒序排序。

#### HTTP Request
GET `/v5/asset/exchange/order-record`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|fromCoin |false |string |兌出幣種，比如 BTC|
|toCoin |false |string |兌入幣種，比如 USDT|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|totalCount |int |返回數據總和 |
|list > fromCoin |string |兌出的幣種|
|list > fromAmount |string |兌出數量/金額 |
|list > toCoin |string |兌入幣種|
|list > toAmount |string |兌入數量/金額|
|list > exchangeRate |string |兌換的匯率 |
|list > createdTime |string |兌換記錄創建時間|
|list > exchangeTxId |string |兌換訂單號|
