升級統一保證金賬戶

#### HTTP Request
POST `/v5/account/upgrade-unified-account`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |


#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result	|string	|用戶賬戶的升級狀態。 1.處理中 2.升級成功 3.升級失敗 4.拒絕
|msg |string |返回的錯誤信息
