該接口可以查詢賬戶下的配置，第一版本是查詢保證金模式。支持 RM 和 PM 模式查詢。


:::caution
未來該接口會增加如下幾個配置查詢：是否是可交易狀態，是否可劃轉狀態等賬戶的配置。
:::


#### HTTP Request
GET `/v5/account/info`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|dcpStatus |string |ON/OFF 枚舉值 查詢當前的time window配置、DCP開關狀態。接口限頻：10次/秒|
|timeWindow |string |斷線計時窗口時間|
|unifiedMarginStatus |string |1：常規未升級統一保證金賬戶。 2：已升級統一保證金賬戶，僅期貨和期權模式。 3：已升級統一保證金賬戶，期貨/期權/現貨統一模式|
|marginMode |string |REGULAR_MARGIN（常規保證金模式） PORTFOLIO_MARGIN（組合保證金模式） |
|updateTime |string |賬戶數據更新的時間，毫秒時間戳|
