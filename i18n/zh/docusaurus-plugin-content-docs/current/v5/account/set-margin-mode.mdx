title: 設置保證金模式
sidebar_label: 設置保證金模式
sidebar_position: 10

用戶如果不設置，默認按常規保證金。該模式僅對 USDC Perp 和 USDC Option 有效，Portfolio Margin 不支持 USDT 結算的合約。

#### HTTP Request
GET `/v5/account/set-margin-mode`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|setMarginMode |**true** |string |REGULAR_MARGIN（常規保證金模式）PORTFOLIO_MARGIN（組合保證金模式）默認常規，傳常規返回設置成功|

| Parameter | Type | Comments|
