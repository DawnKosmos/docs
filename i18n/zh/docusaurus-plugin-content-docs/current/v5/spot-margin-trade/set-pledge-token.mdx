全仓杠杆用户设置保证金币种开关

#### HTTP Request
POST `/v5/spot-margin-trade/set-pledge-token`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin |**true** |string |保证金币种|
|pledgeStatus |**ture** |int |保证金状态：1开启；0关闭|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|pledgeStatus |string |保证金状态：1开启；0关闭 |
