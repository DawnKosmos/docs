全倉槓桿用戶設置保證金幣種開關

#### HTTP Request
POST `/v5/spot-margin-trade/set-pledge-token`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|coin |**true** |string |保證金幣種|
|pledgeStatus |**ture** |int |保證金狀態：1開啟；0關閉|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|pledgeStatus |string |保證金狀態：1開啟；0關閉 |
