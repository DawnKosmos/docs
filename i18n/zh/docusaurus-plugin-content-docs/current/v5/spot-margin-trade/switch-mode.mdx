全倉槓桿開關

#### HTTP Request
GET `/v5/spot-margin-trade/switch-mode`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|switch |**true** |int |1:開啟，0:關閉 |

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|switchStatus |string |全倉槓桿狀態（1:開啟，0:關閉）|
