全倉槓桿設置用戶最大槓桿倍數

#### HTTP Request
GET `/v5/spot-margin-trade/set-leverage`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|leverage |**true**| int |槓桿倍數（2-5）|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status |string |是否調整成功:1:success |
