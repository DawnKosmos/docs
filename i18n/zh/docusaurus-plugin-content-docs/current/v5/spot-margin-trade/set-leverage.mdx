全仓杠杆设置用户最大杠杆倍数

#### HTTP Request
GET `/v5/spot-margin-trade/set-leverage`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|leverage |**true**| int |杠杆倍数（2-5）|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|status |string |是否调整成功:1:success |
