获取当前统一保证金账户的抵押品信息，包括借贷利率，可借贷金额以及抵押品折算率，是否可抵押作为保证金等信息。

#### HTTP Request
GET `/v5/account/collateral-info`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|currency |false |string |目前所有抵押品的资产币种|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|list > currency |string |目前所有抵押品的资产币种|
|list > hourlyBorrowRate |string |每小时借款利率|
|list > maxBorrowAmount |string |最大可借贷额度|
|list > freeBorrowAmount |string |免息借贷额度
|list > borrowAmount |string |已用借贷额度
|list > availableToBorrow |string |用户剩余可借额度
|list > borrowable |bool |是否是可借贷的借贷币种 : true 是; false 否
|list > marginCollateral |bool |是否可作为保证金抵押币种: true 是; false 否
|list > collateralStatus |string |是否能作为抵押品的状态：1 是；0 否
|list > collateralRatio |string |抵押品币种折算为保证金的折算率
