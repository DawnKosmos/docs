升级统一保证金账户

#### HTTP Request
POST `/v5/account/upgrade-unified-account`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |


#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|result	|string	|用户账户的升级状态。1.处理中 2.升级成功 3.升级失败 4.拒绝
|msg |string |返回的错误信息
