Query the qty and amount of borrowable coin in spot account.
> **Cover: Spot (Unified Account)**

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### HTTP Request
GET `/v5/order/spot-borrow-check`

### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|[category](../enum#category) |**true** |string |Product type. `spot`,`cross-margin` |
|symbol |**true** |string |Symbol name|
|side |**true** |string |Transaction side. `Buy`,`Sell` |

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|------ |
|symbol |string |Symbol name |
|side |string |Side |
|maxBorrowableQty |string |The maximum qty of borrowable |
|maxBorrowableAmount |string |The maximum amount of borrowable |
|borrowCoin |string |Borrow coin |
