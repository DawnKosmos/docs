升級統一保證金賬戶

:::note important
封禁 / 場外借貸 / 淨資產不滿足 / 極簡鏈路用戶當前無法升級至統一帳戶.
:::

### HTTP 請求
POST `/v5/account/upgrade-to-uta`

### 請求參數
None

### 響應參數
| Parameter | Type | Comments|
|:----- |:-----|----- |
|unifiedUpdateStatus |string |用戶賬戶的升級狀態。 `1`: 處理中, `2`: 升級成功, `3`: 升級失敗, `4`: 拒絕 |
|unifiedUpdateMsg |array |Object |
|msg |string |返回的錯誤信息 |
