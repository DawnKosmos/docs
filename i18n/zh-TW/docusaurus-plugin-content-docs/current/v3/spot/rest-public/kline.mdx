:::info
返回指定的時間間隔最近 3500 根的K線結果
:::

:::info
如果startTime和endTime沒有發送，只有最新的K線會被返回
:::

#### HTTP 請求
GET <code>/spot/v3/public/quote/kline</code>

#### 請求參數
| 參數 | 是否必須 | 類型 | |
|:----- |:-------|:-----|------ |
| symbol | **true** | string | 幣對名稱 |
| interval | **true** | string | K線圖區間。可識別發送的值為：1m, 5m, 15m, 30m, 1h, 1d, 1w, 1M (m=分鐘, h=小時, d=天, w=星期, M=月)|
| limit | false | integer | 默認1000， 最大1000 |
| startTime | false | long | 第一條返回的時間戳(毫秒) |
| endTime | false | long | 最後一條返回的時間戳(毫秒) |

#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|t| long | 時間戳(毫秒) |
|s| string | 幣對名稱 |
|sn| string | 幣對別名 |
|c| string | 收盤價 |
|h| string | 最高價 |
|l| string | 最低價 |
|o| string | 開盤價 |
|v| string | 交易金額 |

### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/kline?symbol=BTCUSDT&interval=1m&limit=1'
```

</TabItem>
</Tabs>
