#### HTTP 請求
GET <code>/spot/v3/public/quote/trades</code>

#### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
| symbol | **true** | string | 幣對名稱 |
| limit | false | integer | 默認60，最大60 |

#### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| price | string | 價格 |
| time | long | 當前Unix時間戳，毫秒(ms) |
| qty | string | 交易數量 |
| isBuyerMaker | integer | `0`：買單, `1`：賣單 |

### 請求示例

<Tabs>
<TabItem value="console" label="Console">

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/public/quote/trades?symbol=BTCUSDT&limit=1'
```

</TabItem>
</Tabs>
