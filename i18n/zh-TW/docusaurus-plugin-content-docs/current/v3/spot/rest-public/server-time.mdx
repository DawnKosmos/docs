### HTTP 請求
GET <code>/spot/v3/public/server-time</code>

### 請求參數
無

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
| serverTime | string | Bybit服務器時間戳 (毫秒) |

### 請求示例

```http
curl https://api-testnet.bybit.com/spot/v3/public/server-time
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "OK",
    "result": {
        "serverTime": "1670521545784"
    },
    "retExtInfo": {},
    "time": 1670521545785
}
```
