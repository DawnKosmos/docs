### HTTP 請求
GET <code>/spot/v3/private/cross-margin-account</code>

### 請求參數
無

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|acctBalanceSum|string |總持有BTC折合|
|debtBalanceSum|string |总负债BTC折合|
|loanAccountList|array |Object|
|> free|string|可用|
|> interest|string |未還利息 |
|> loan|string |未還本金 |
|> locked|string |鎖定金額 |
|> tokenId|string |幣種 |
|> total|string|總計 |
|riskRate|string|風險率 |
|status|integer |賬戶狀態（`1`: 正常, `2`: 限制提幣/劃轉, `3`: 預警, `4`: 已爆倉, `5`: 已穿倉）|

### 請求示例

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/private/cross-margin-account' \
--header 'X-BAPI-SIGN-TYPE: 2' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662619341689' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXXX'
```
