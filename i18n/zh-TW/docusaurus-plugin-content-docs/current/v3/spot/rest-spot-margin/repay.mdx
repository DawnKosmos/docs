### HTTP 請求
GET <code>/spot/v3/private/cross-margin-repay</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|coin|<b>true</b>|string|幣種|
|qty|<b>true</b>|string|歸還數量|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|repayId|string|還款id|

### 請求示例

```http
curl --location --request POST 'https://api-testnet.bybit.com/spot/v3/private/cross-margin-repay' \
--header 'Content-Type: application/json' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662618298354' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXXX' \
--data-raw '{
    "coin": "USDT",
    "qty": "150"
}'
```
