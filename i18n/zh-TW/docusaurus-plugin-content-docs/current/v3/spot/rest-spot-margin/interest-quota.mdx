### HTTP 請求
GET <code>/spot/v3/private/cross-margin-loan-info</code>

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|coin|<b>true</b>|string|幣種|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|coin|string|幣種|
|interestRate|string|基礎日利率|
|loanAbleAmount|string|用戶剩餘可借金額|
|maxLoanAmount|string|請忽略該字段|

### 請求示例

```http
curl --location --request GET 'https://api-testnet.bybit.com/spot/v3/private/cross-margin-loan-info?coin=USDT' \
--header 'X-BAPI-SIGN-TYPE: 2' \
--header 'X-BAPI-API-KEY: XXXXXXXXX' \
--header 'X-BAPI-TIMESTAMP: 1662619710952' \
--header 'X-BAPI-RECV-WINDOW: 5000' \
--header 'X-BAPI-SIGN: XXXXX'
```
