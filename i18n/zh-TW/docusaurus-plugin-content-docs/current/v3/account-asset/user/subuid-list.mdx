獲取當前母帳戶下所有的子帳戶列表。需使用<b>母</b>帳戶的API key。

:::tip
在調用接口時，使用的API key至少需要擁有以下其中一種權限
* 母子API key: "Account Transfer（資產帳戶劃轉）", "Subaccount Transfer（母子帳戶劃轉）", "Withdrawal（提幣）"
:::

### HTTP 請求
POST `/user/v3/private/query-sub-members`

### 請求參數
無

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|subMembers |array |Object |
|uid |string |子帳戶userId |
|username |string |用戶名|
|memberType |integer |`1`: 普通子帳戶, `6`: 託管子帳戶 |
|status |integer |帳戶狀態.<ul><li>`1`: 正常</li><li>`2`: 登陸封禁</li><li>`4`: 凍結 </li></ul>|
|remark |string |備註 |

### 請求示例

```http
POST /user/v3/private/create-sub-member HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: xxxxxxxxxxxx
X-BAPI-API-KEY: xxxxxxxxxxxx
X-BAPI-TIMESTAMP: 1671174837503
X-BAPI-RECV-WINDOW: 50000
Content-Type: application/json
Content-Length: 109

{
    "username": "testsubaccount",
    "memberType": 1,
    "switch": 1,
    "note": "testnet subaccount"
}
```
