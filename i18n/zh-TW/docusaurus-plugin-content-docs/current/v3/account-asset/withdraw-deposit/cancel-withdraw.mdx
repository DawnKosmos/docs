撤銷提幣

### HTTP 請求
POST `/asset/v3/private/withdraw/cancel`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|id| <b>true</b> | int64 |提現ID。 (提現API響應參數中的提現請求ID)|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|status|int|結果對象（撤單申請結果：1成功，0失敗）: <ul><li>`1` 成功</li><li>`0` 失敗</li></ul>|

```http
X-BAPI-RECV-WINDOW: 50000

Content-Length: 24

{
    "id": 1234567890
}
```
