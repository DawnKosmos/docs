賬號內可劃轉幣種列表

此接口只會返回使用萬能劃轉 API 進行的劃轉。 不會返回由其他接口進行的劃轉。

### HTTP 請求
GET `/asset/v3/private/transfer/transfer-coin/list/query`

### 請求參數
| 參數 | 是否必須 | 類型 | 說明|
|:----- |:-------|:-----|------ |
|[fromAccountType](../enum#accounttype)|<b>true</b> |string |劃出賬戶類型. 劃入劃出帳戶類型必須不一樣|
|[toAccountType](../enum#accounttype) |<b>true</b> |string |劃出賬戶類型. 劃入劃出帳戶類型必須不一樣|

### 返回參數
| 參數 | 類型 | 說明|
|:----- |:-----|----- |
|list |Array |賬號內互轉支持的幣種|
