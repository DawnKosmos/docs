Subscribe execution stream.

:::tip
You may have multiple pushes for one order.
:::

**Topic:**<br/>
`user.execution.unifiedAccount`

### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|id |string |Message id |
|topic |string |Topic name |
|type |string |Message type. `snaptshot`|
|ts |number |The timestamp (ms) that system generates the data. |
|data |Object | |
|> version |integer |Version|
|> result |array |Object|
|>> symbol |string |Symbol name |
|>> orderId |string |Order id |
|>> side |string |Side. `Buy`,`Sell` |
|>> orderLinkId |string |User customized order id |
|>> orderPrice |string |Order price |
|>> orderQty |string |Order qty |
|>> orderType |string |Order type |
|>> execId |string |Execution id |
|>> execPrice |string |Execution price |
|>> execQty |string |Execution qty |
|>> execValue |string |Executed order value |
|>> feeRate |string |Trading fee rate |
|>> execFee |string |Executed trading fee |
|>> execType |string |成交類型. `TRADE`,`ADLTRADE`,`FUNDING`,`BUSTTRADE`,`SETTLE`|
|>> [lastLiquidityInd](../enum#lastliquidityind) |string |流動性類型. `TAKER`: 吃流動性,`MAKER`: 增加流動性 |
|>> leavesQty |string |The remaining qty not executed |
|>> [stopOrderType](../enum#stopordertype) |string |Stop order type. If the order is not stop order, any type is not returned |
