#### HTTP Request
GET `/derivatives/v3/public/funding/history-funding-rate`

#### Request Parameters
| Parameter | Required | Type | Comments|
|:----- |:-------|:-----|------ |
|category |false |string |Products category. `future` ONLY |
|symbol |<b>true</b> |string |Symbol name|
|start |false |string |The start timestamp (ms)|
|end |false |string |The end timestamp (ms)|
|limit |false |string |Limit for data size per page. [1,200]. default: `200`|

#### Response Parameters
| Parameter | Type | Comments|
|:----- |:-----|----- |
|category |string |Products category. Keeps empty string `""` when `category` is not passed |
|list |array |Object |
|> symbol |string |Symbol name |
|> fundingRate |string |Funding rate |
|> fundingRateTimestamp |string |Funding rate timestamp(ms) |
