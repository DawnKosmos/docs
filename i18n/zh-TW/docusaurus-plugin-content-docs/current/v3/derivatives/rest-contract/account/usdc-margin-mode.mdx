該接口可以查詢USDC賬戶下的配置

### HTTP 請求
GET `/contract/v3/private/account/info`

### 請求參數
無

### 響應參數
| 參數 | 類型 | 說明|
|:----- |:-----|------ |
|marginMode |string |保證金模式. `REGULAR_MAEGIN`: 普通保證金,`PORTFOLIO_MARGIN`: 組合保證金 |
|updateTime |string |帳戶數據更新時間戳 (毫秒) |

### 請求示例

```http
GET /contract/v3/private/account/info HTTP/1.1
Host: api-testnet.bybit.com
X-BAPI-SIGN: XXXXX
X-BAPI-API-KEY: XXXXX
X-BAPI-TIMESTAMP: 1670941248049
X-BAPI-RECV-WINDOW: 5000
```

### 響應示例

```json
{
    "retCode": 0,
    "retMsg": "Success.",
    "result": {
        "marginMode": "REGULAR_MARGIN",
        "updateTime": "1669888247000"
    }
}
```
